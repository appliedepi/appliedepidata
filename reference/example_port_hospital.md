# Port Hospital subset, fictional Ebola Virus Disease case linelist (West Africa, 2014)

Port Hospital subset of the cleaned fictional Ebola Virus Disease case
linelist, one of six per-hospital files prepared for an iteration/loops
example where multiple files are read in and combined.

## Usage

``` r
example_port_hospital
```

## Format

A data frame with 1762 rows and 30 columns

## Source

Within this package the original dataset is `example_port_hospital.csv`.

This is version 1.0.

This dataset is English.

## Details

Row subset of
[example_hospital_linelists](https://appliedepi.github.io/appliedepidata/reference/example_hospital_linelists.md)
/
[hospital_linelists](https://appliedepi.github.io/appliedepidata/reference/hospital_linelists.md)
filtered to cases reported at Port Hospital. See also
[example_central_hospital](https://appliedepi.github.io/appliedepidata/reference/example_central_hospital.md),
[example_military_hospital](https://appliedepi.github.io/appliedepidata/reference/example_military_hospital.md),
[example_st_marks_maternity_hospital](https://appliedepi.github.io/appliedepidata/reference/example_st_marks_maternity_hospital.md),
[example_missing](https://appliedepi.github.io/appliedepidata/reference/example_missing.md),
and
[example_other](https://appliedepi.github.io/appliedepidata/reference/example_other.md).
