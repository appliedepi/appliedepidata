# Feveriosis surveillance data from Feveria from Jan 1 2025 to Aug 31 2026

Surveillance data for fictional disease feveriosis. Reported in
fictional country Feveria in 2025 & 2026.

## Usage

``` r
feveriosis_main
```

## Format

A dataframe with 449 rows and 13 columns.

## Source

Fictional data created by Applied Epi for training purposes

This is version 1.1

This dataset is English.

## Details

This is a linelist with fictional case-level data from the imaginary
country Feveria, with clinical and epidemiological information for cases
of feveriosis (fictional disease). The data is intended for use as part
of the case study "Analysis of routine surveillance of feveriosis"
alongside
[feveriosis_recent](https://appliedepi.github.io/appliedepidata/reference/feveriosis_recent.md)
(linelist of feveriosis cases from Aug 31 - Sep 4, 2026 in Feveria).
