# Model run summary, fictional Ebola outbreak in Sierra Leone (2014) - epidemic models example

Summary table, unbundled from
[epidemic_models_epinow_res](https://appliedepi.github.io/appliedepidata/reference/epidemic_models_epinow_res.md)
(`$summary`) for the epidemic models example.

## Usage

``` r
epidemic_models_summary
```

## Format

A data frame with 5 rows and 3 columns

## Source

The original dataset is epidemic_models_summary.rds

This is version 1.0.

This dataset is English.
