# Test results for notifiable disease surveillance cases in Feveria in 2024

Test results for notifiable disease surveillance cases notified in
fictional country Feveria in 2024, for malaria, dengue, typhoid, yellow
fever, and cholera.

## Usage

``` r
multidisease_tests
```

## Format

A dataframe with 1314 rows and 7 columns.

## Source

Fictional data created by Applied Epi for training purposes

This is version 1.0.

This dataset is English.

## Details

This is a linelist with fictional test-level data from the imaginary
country Feveria, with laboratory result information for suspected cases
of malaria, dengue, typhoid, yellow fever, and cholera. The data is
intended for use as part of the case study "Linking and analyzing
notifiable disease surveillance data and laboratory data in R" alongside
[multidisease_notifications](https://appliedepi.github.io/appliedepidata/reference/multidisease_notifications.md)
(notification data in Feveria).
