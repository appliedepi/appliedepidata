# Latest observation date, fictional Ebola outbreak in Sierra Leone (2014) - epidemic models example

Latest observation date, unbundled from
[epidemic_models_epinow_res](https://appliedepi.github.io/appliedepidata/reference/epidemic_models_epinow_res.md)
(`$estimates$observations`) for the epidemic models example.

## Usage

``` r
epidemic_models_latest_date
```

## Format

A length-1 Date vector

## Source

The original dataset is epidemic_models_latest_date.rds

This is version 1.0.

This dataset is English.
