# Linelist data to accompany next generation sequencing data of Shigella surveillance samples from Belgium 2013-2019

Linelist data to accompany next generation sequencing data of Shigella
surveillance samples from Belgium 2013-2019

## Usage

``` r
sample_data_Shigella_tree
```

## Format

A dataframe with 299 rows and 13 columns.

## Source

This data was posted on the [European Nucleotide
Archive](https://www.ebi.ac.uk/ena/browser/view/PRJEB40097?show=xrefs)

The original dataset is sample_data_Shigella_Tree.txt

This is version 1.0.

This dataset is English.

4.0](https://creativecommons.org/licenses/by-nc-sa/4.0/)

## Details

This linelist data to accompany sequencing data from routine
surveillance samples collected in Belgium between 2013 and 2019. It can
be used together with
[Shigella_tree](https://appliedepi.github.io/appliedepidata/reference/Shigella_tree.md)

## References

Fischer, N. et al. Genomic epidemiology of persistently circulating MDR
Shigella sonnei strains associated with men who have sex with men (MSM)
in Belgium (2013-19)
[doi:10.1093/jac/dkab377](https://doi.org/10.1093/jac/dkab377)
