# Recent feveriosis surveillance data from Viraland from Aug 31 2026 to Sep 4 2026 \\Russian\\

Russian version of the recent surveillance data for fictional disease
feveriosis. Reported in fictional country Viraland from August 31, 2026
to September 4, 2026.

## Usage

``` r
feveriosis_recent_ru
```

## Format

A dataframe with 28 rows and 12 columns.

## Source

Fictional data created by Applied Epi for training purposes

This is version 2.1

This dataset is Russian.

## Details

This is a linelist of a single week of case-level data of feveriosis
(fictional data) from the imaginary country Viraland. The data is
intended for use as part of the Russian version of the case study
"Analysis of routine surveillance of feveriosis" alongside the Russian
[feveriosis_main_ru](https://appliedepi.github.io/appliedepidata/reference/feveriosis_main_ru.md)
(compiled linelist of feveriosis cases from Jan 1 2025 - Aug 31, 2026 in
Viraland).
