# Posterior estimated-reported-cases samples, fictional Ebola outbreak in Sierra Leone (2014) - epidemic models example

Posterior samples of estimated reported cases, unbundled from
[epidemic_models_epinow_res](https://appliedepi.github.io/appliedepidata/reference/epidemic_models_epinow_res.md)
(`$estimated_reported_cases$samples`) for the epidemic models example.

## Usage

``` r
epidemic_models_estimated_reported_cases_samples
```

## Format

A data frame with 308320 rows and 4 columns

## Source

The original dataset is
epidemic_models_estimated_reported_cases_samples.rds

This is version 1.0.

This dataset is English.

## Details

See
[epidemic_models_epinow_res](https://appliedepi.github.io/appliedepidata/reference/epidemic_models_epinow_res.md)
for the full model output this was extracted from.
