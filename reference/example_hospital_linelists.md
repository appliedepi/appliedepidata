# Combined hospital linelists example, fictional Ebola Virus Disease data (West Africa, 2014)

Excel file with the combined (all-hospital) cleaned version of fictional
case linelist data simulating properties from the West African Ebola
outbreak of 2014, prepared for an iteration/loops example folder.

## Usage

``` r
example_hospital_linelists
```

## Format

A data frame with 454 rows and 30 columns

## Source

This data is a cleaned version of
[example_case_linelist_2020_10_08](https://appliedepi.github.io/appliedepidata/reference/example_case_linelist_2020_10_08.md).
Within this package the original dataset is
example_hospital_linelists.xlsx.

This is version 1.0.

This dataset is English.

## Details

Same content as
[hospital_linelists](https://appliedepi.github.io/appliedepidata/reference/hospital_linelists.md).
Intended to be compared against the per-hospital splits
[example_central_hospital](https://appliedepi.github.io/appliedepidata/reference/example_central_hospital.md),
[example_military_hospital](https://appliedepi.github.io/appliedepidata/reference/example_military_hospital.md),
[example_port_hospital](https://appliedepi.github.io/appliedepidata/reference/example_port_hospital.md),
[example_st_marks_maternity_hospital](https://appliedepi.github.io/appliedepidata/reference/example_st_marks_maternity_hospital.md),
[example_missing](https://appliedepi.github.io/appliedepidata/reference/example_missing.md),
and
[example_other](https://appliedepi.github.io/appliedepidata/reference/example_other.md),
which together reconstruct this same data when row-bound.
