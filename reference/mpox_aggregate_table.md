# Fictional cumulative mpox case totals from five European countries, 2022

Fictional case counts based on the real emergence of Mpox in Europe in
2022.

## Usage

``` r
mpox_aggregate_table
```

## Format

An object of class `data.frame` with 101 rows and 3 columns.

## Source

Fictional data provided by ECDC EI Group for training purposes

This is version 1.0.

This dataset is English.

## Details

This is a table with cumulative mpox case totals by date and country
(country A - E). It is intended for use as part of the Mpox case study,
alongside
[mpox_linelist](https://appliedepi.github.io/appliedepidata/reference/mpox_linelist.md)
(case linelist with fictional case-level information data from five
countries).
