# Fictional mpox linelist data from five European countries, 2022

Fictional linelist data based on the real emergence of mpox in Europe in
2022.

## Usage

``` r
mpox_linelist
```

## Format

An object of class `data.frame` with 2000 rows and 13 columns.

## Source

Fictional data provided by ECDC EI Group for training purposes

This is version 1.0.

This data is English.

4.0](https://creativecommons.org/licenses/by-nc-sa/4.0/)

## Details

This is a case linelist which contains fictional individual level data
from five countries (country A - E) on clinical, laboratory and
epidemiological information. It is intended for use as part of the mpox
case study, alongside
[mpox_aggregate_table](https://appliedepi.github.io/appliedepidata/reference/mpox_aggregate_table.md)
(cumulative case totals by date and country).
