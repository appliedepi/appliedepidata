# Fictional counts of malaria cases from Mozambique in 2019

Fictional counts of malaria cases from Mozambique in 2019

## Usage

``` r
malaria_facility_count_data
```

## Format

A data frame with 3038 rows and 10 columns

## Source

This data is fictional based on data from Mozambique. It has been
adapted from
[district_weekly_count_data](https://appliedepi.github.io/appliedepidata/reference/district_weekly_count_data.md).

The original dataset malaria_facility_count_Data.rds.

This is version 1.0.

This dataset is English.

4.0](https://creativecommons.org/licenses/by-nc-sa/4.0/)

## Details

This is fictional aggregate data with counts by date, region and age
group.
