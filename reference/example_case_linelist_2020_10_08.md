# Dated snapshot of fictional case linelist data simulating properties from the West African Ebola outbreak of 2014

A dated snapshot (2020-10-08) of fictional case linelist data simulating
properties from the West African Ebola outbreak of 2014, prepared for an
iteration/loops example folder.

## Usage

``` r
example_case_linelist_2020_10_08
```

## Format

A data frame with 6611 rows and 28 columns

## Source

This data was originally added to the outbreaks package, extensive edits
have been made for teaching purposes. Within this package the original
dataset is example_case_linelist_2020-10-08.xlsx.

This is version 1.0.

This dataset is English.
