# Outputs from epinow2, fictional Ebola outbreak in Sierra Leone (2014) - epidemic models example

Outputs from epinow2 based on fictional Ebola Virus Disease data
simulating properties from the West African Ebola outbreak of 2014,
repackaged for the epidemic models example.

## Usage

``` r
epidemic_models_epinow_res

epidemic_models_epinow_res_small

epidemic_models_generation_time

epidemic_models_incubation_period
```

## Format

A large list

## Source

The original dataset is epidemic_models_epinow_res.rds

This is version 1.0.

This dataset is English.

This dataset is licensed under [MIT](https://opensource.org/license/MIT)

## Details

These outputs are produced using
[linelist_cleaned_rds](https://appliedepi.github.io/appliedepidata/reference/linelist_cleaned_rds.md)
with [epinow2](https://epiforecasts.io/EpiNow2/). Same content as
[epinow_res](https://appliedepi.github.io/appliedepidata/reference/epinow_res.md).
