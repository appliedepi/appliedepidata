# Fictional case linelist data simulating properties from the West African Ebola outbreak of 2014

Fictional case linelist data simulating properties from the West African
Ebola outbreak of 2014

## Usage

``` r
case_linelists_linelist_raw
```

## Format

A data frame with 6611 rows and 28 columns

## Source

This data was originally added to the outbreaks package, extensive edits
have been made for teaching purposes. The data was originally simulated
to match properties of the West African Ebola outbreak of 2014-2015 (see
outbreaks package documentation for details).

The original dataset can be accessed using outbreaks::ebola_sim. Within
this package the original dataset is case_linelists_linelist_raw.xlsx.

This is version 1.0.

This dataset is English.

This dataset is licensed under [MIT](https://opensource.org/license/MIT)

## Details

This is fictional case-linelist which contains de-indentified individual
level data on clinical outcome, demographics and region. Same content as
[linelist_raw](https://appliedepi.github.io/appliedepidata/reference/linelist_raw.md),
repackaged under the `case_linelists_` naming for this bundle.
