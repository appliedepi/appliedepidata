# Linelist data of Campylobacter surveillance in Germany 2001-2011.

Linelist data of Campylobacter surveillance in Germany 2001-2011.

## Usage

``` r
campylobacter_germany
```

## Format

A data frame with 509 rows and 2 columns

## Source

This data was originally added to the surveillance package, we have
removed the last 13 rows of data and only kept date and cases variables.
The data on campylobacteriosis cases have been queried from the
"SurvstatRKI" database of the German Robert Koch Institute
(https://survstat.rki.de/).

The original dataset can be accessed using data("CampyDe"). Within this
package the original dataset is campylobacter_germany.xlsx

This is version 1.0.

This dataset is English.

This dataset is licensed under
[GPL3](https://www.gnu.org/licenses/gpl-3.0.html)

## Details

Weekly number of reported campylobacteriosis cases in Germany,
2002-2011.
