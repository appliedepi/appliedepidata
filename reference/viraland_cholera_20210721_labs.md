# Lab data with cholera test results for Viraland, 21 July 2021

Raw surveillance data for a cholera outbreak in fictional country
Viraland in 2021, with data as of 21 July.

## Usage

``` r
viraland_cholera_20210721_labs
```

## Format

A dataframe with 45 rows and 6 columns.

## Source

Fictional data created by Applied Epi for training purposes

This is a derivate (subset) of the clean cholera surveillance data for
Viraland, 21 July 2021.

This dataset is English.

## Details

This is a linelist with fictional cholera test-level data from the
imaginary country Viraland. The data is intended for use alongside the
intermediate qmd course and the intro to R course.
