# Phylogenetic tree produced from next generation sequencing data of Shigella surveillance samples from Belgium 2013-2019

Outputs of analysis with next generation sequencing data of Shigella
surveillance samples from Belgium 2013-2019

## Usage

``` r
Shigella_subtree_2
```

## Format

A List of class "phylo"

## Source

This data was posted on the [European Nucleotide
Archive](https://www.ebi.ac.uk/ena/browser/view/PRJEB40097?show=xrefs)

The original dataset is Shigella_subtree_2.txt

This is version 1.0.

This dataset is English.

This dataset is licensed under [CC by-NC-SA
4.0](https://creativecommons.org/licenses/by-nc-sa/4.0/)

## Details

This is based on sequencing data from routine surveillance samples
collected in Belgium between 2013 and 2019. It was produced using used
together with
[sample_data_Shigella_tree](https://appliedepi.github.io/appliedepidata/reference/sample_data_Shigella_tree.md)
and
[Shigella_tree](https://appliedepi.github.io/appliedepidata/reference/Shigella_tree.md)

## References

Fischer, N. et al. Genomic epidemiology of persistently circulating MDR
Shigella sonnei strains associated with men who have sex with men (MSM)
in Belgium (2013-19)
