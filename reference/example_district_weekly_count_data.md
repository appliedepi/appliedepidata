# Fictional counts of malaria cases from Mozambique (example copy)

Fictional counts of malaria cases from Mozambique, repackaged for an
iteration/loops example folder.

## Usage

``` r
example_district_weekly_count_data
```

A data frame with 9628 rows and 4 columns
