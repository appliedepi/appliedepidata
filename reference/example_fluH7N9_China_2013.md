# Linelist data from an outbreak of influenza A H7N9 in China 2013 (example copy)

Linelist data from an outbreak of influenza A H7N9 in China 2013,
repackaged for an iteration/loops example folder.

## Usage

``` r
example_fluH7N9_China_2013
```

## Format

A data frame with 136 rows and 8 columns
