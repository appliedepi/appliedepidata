# Notifiable disease surveillance data from Feveria in 2024 \\French\\

French version of the notifiable disease surveillance data for malaria,
dengue, typhoid, yellow fever, and cholera. Reported in fictional
country Feveria in 2024.

## Usage

``` r
multi_maladies_notifications
```

## Format

A dataframe with 987 rows and 12 columns.

## Source

Fictional data created by Applied Epi for training purposes

This is version 1.0.

This dataset is French.

## Details

This is a linelist with fictional notification-level data from the
imaginary country Feveria, with clinical and epidemiological information
for suspected cases of malaria, dengue, typhoid, yellow fever, and
cholera. The data is intended for use as part of the French version of
the case study "Linking and analyzing notifiable disease surveillance
data and laboratory data in R" alongside the French
[multi_maladies_tests](https://appliedepi.github.io/appliedepidata/reference/multi_maladies_tests.md)
(laboratory test results for notifications in Feveria).
