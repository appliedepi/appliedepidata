# Search and Filter Datasets Available in appliedepidata

Search and Filter Datasets Available in appliedepidata

## Usage

``` r
search_data()
```

## Value

A Shiny application object. This function will open the Shiny app in a
web browser when executed.

## Details

Launches an interactive Shiny application that allows users to filter
and search a dataset. Users can select a language from a dropdown menu
and enter a search query to filter the data displayed in a reactive
table.

## Examples

``` r
if (FALSE) { # \dontrun{
# Launch the interactive data search application
search_data()
} # }
```
