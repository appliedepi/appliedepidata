# Age/sex population proportions for Sierra Leone (GIS basics example)

Age/sex population proportions for Sierra Leone, for the GIS basics
population pyramid example.

## Usage

``` r
gis_population_sle_population_statistics_sierraleone_2020
```

## Format

A data frame with 3 rows and 18 columns

## Source

This population data was downloaded from [Humanitarian Data
Exchange](https://data.humdata.org/dataset/cod-ps-sle). No edits were
made to this.

Within this package the original dataset is
gis_population_sle_population_statistics_sierraleone_2020.xlsx

This is version 1.0.

This dataset is English.

## Details

Same source as
