# Weather data for Germany from 2001-2011

Weather data for Germany from 2001-2011

## Usage

``` r
germany_weather
```

## Format

A list

## Source

This data was obtained using the ecmwfr package as described in the
epiRhandbook chapter on timeseries. This data was originally collected
with the European Union Copernicus satelite and made available on their
[climate data
store](https://cds.climate.copernicus.eu/datasets/derived-era5-land-daily-statistics?tab=overview)

The original dataset is germany_weather.zip

This is version 1.0.

This dataset is English.
