# Subset with additional spatial data of cleaned fictional Ebola Virus Disease data simulating properties from the West African Ebola outbreak of 2014

Subset with additional spatial data of rds clean version of fictional
Ebola Virus Disease data simulating properties from the West African
Ebola outbreak of 2014

## Usage

``` r
linelist_cleaned_with_adm3
```

## Format

A data frame with 454 rows and 30 columns

## Source

This data is a cleaned version of
[linelist_cleaned_rds](https://appliedepi.github.io/appliedepidata/reference/linelist_cleaned_rds.md),
which is cleaned using code in the [epiRhandbook cleaning
chapter](https://epirhandbook.com/en/new_pages/cleaning.html).

The original dataset is linelist_cleaned_with_adm3.rds

This is version 4.0.

This dataset is English.

This dataset is licensed under [MIT](https://opensource.org/license/MIT)

## Details

This is fictional case-linelist which contains de-indentified individual
level data on clinical outcome, demographics and region.

## References

WHO Ebola Response Team. 2014. Ebola virus disease in West Africa–the
first 9 months of the epidemic and forward projections. The New England
journal of medicine 371:1481–1495
