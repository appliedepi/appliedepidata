# Outputs from epinow2 based on fictional Ebola Virus Disease data simulating properties from the West African Ebola outbreak of 2014

Outputs from epinow2 based on fictional Ebola Virus Disease data
simulating properties from the West African Ebola outbreak of 2014

## Usage

``` r
epinow_res

epinow_res_small

generation_time

incubation_period
```

## Format

A large list

An object of class `epinow` (inherits from `list`) of length 3.

An object of class `list` of length 5.

An object of class `list` of length 5.

## Source

This is direct output from use with
[linelist_cleaned_rds](https://appliedepi.github.io/appliedepidata/reference/linelist_cleaned_rds.md).

The original datasets are epinow_res.rds, epinow_res_small.rds,
generation_time.rds and incubation_period.rds

This is version 1.0.

This dataset is English.

This dataset is licensed under [MIT](https://opensource.org/license/MIT)

## Details

These outputs are produced using
[linelist_cleaned_rds](https://appliedepi.github.io/appliedepidata/reference/linelist_cleaned_rds.md)
with [epinow2](https://epiforecasts.io/EpiNow2/)
