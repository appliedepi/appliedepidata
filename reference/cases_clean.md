# Fictional case data for a generic disease to demonstrate contact tracing

Fictional case data for a generic disease to demonstrate contact tracing

## Usage

``` r
cases_clean

contacts_clean

followups_clean

relationships_clean
```

## Format

A data frame with 56 rows and 86 columns

An object of class `data.frame` with 66 rows and 62 columns.

An object of class `data.frame` with 437 rows and 53 columns.

An object of class `data.frame` with 100 rows and 21 columns.

## Source

This data was originally sample contact tracing data from the
[Go.Data](https://worldhealthorganization.github.io/godata/) platform

The original datasets are cases_clean.rds, contacts_clean.rds,
followups_clean.rds, relationships_clean.rds

This is version 1.0.

This dataset is English.

This dataset is licensed under
[GPL3](https://www.gnu.org/licenses/gpl-3.0.html)

## Details

This is fictional case-linelist which contains de-indentified individual
level data, based on go.data format.
