# Population data at admin level 3 for Sierra Leone from Humanitarian Data Exchange

Population data at admin level 3 for Sierra Leone from Humanitarian Data
Exchange

## Usage

``` r
sle_admpop_adm3_2020
```

## Format

A shapefile with 167 rows and 20 columns

## Source

This population data was downloaded from [Humanitarian Data
Exchange](https://data.humdata.org/dataset/cod-ps-sle) No edits were
made to this.

This is the original dataset is sle_admpop_adm3_2020.csv

This is version 1.0.

This dataset is English.

This dataset is licensed under [CC
by-IGO](https://data.humdata.org/faqs/licenses)

## Details

This population data can be used with
[linelist_cleaned_with_adm3](https://appliedepi.github.io/appliedepidata/reference/linelist_cleaned_with_adm3.md)
and
[sle_adm3](https://appliedepi.github.io/appliedepidata/reference/sle_adm3.md).
