# Reported case counts, fictional Ebola outbreak in Sierra Leone (2014) - epidemic models example

Observed case counts, unbundled from
[epidemic_models_epinow_res](https://appliedepi.github.io/appliedepidata/reference/epidemic_models_epinow_res.md)
(`$estimates$observations`) for the epidemic models example.

## Usage

``` r
epidemic_models_reported_cases
```

## Format

A data frame with 367 rows and 2 columns

## Source

The original dataset is epidemic_models_reported_cases.rds

This is version 1.0.

This dataset is English.
