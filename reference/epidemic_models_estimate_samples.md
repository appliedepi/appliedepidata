# Posterior estimate samples, fictional Ebola outbreak in Sierra Leone (2014) - epidemic models example

Posterior samples of the epinow2 estimates, unbundled from
[epidemic_models_epinow_res](https://appliedepi.github.io/appliedepidata/reference/epidemic_models_epinow_res.md)
(`$estimates$samples`) for the epidemic models example.

## Usage

``` r
epidemic_models_estimate_samples
```

## Format

A data frame with 1242724 rows and 8 columns

## Source

The original dataset is epidemic_models_estimate_samples.rds

This is version 1.0.

This dataset is English.

## Details

See
[epidemic_models_epinow_res](https://appliedepi.github.io/appliedepidata/reference/epidemic_models_epinow_res.md)
