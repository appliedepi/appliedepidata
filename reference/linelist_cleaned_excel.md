# Cleaned fictional Ebola Virus Disease data simulating properties from the West African Ebola outbreak of 2014

Excel file with cleaned version of fictional Ebola Virus Disease data
simulating properties from the West African Ebola outbreak of 2014

## Usage

``` r
linelist_cleaned_excel
```

## Format

A data frame with 5888 rows and 30 columns

## Source

This data is a cleaned version of
[linelist_raw](https://appliedepi.github.io/appliedepidata/reference/linelist_raw.md),
which is cleaned using code in the [epiRhandbook cleaning
chapter](https://epirhandbook.com/en/new_pages/cleaning.html).

The original dataset is linelist_cleaned_excel.xlsx

This is version 2.0.

This dataset is English.

This dataset is licensed under [MIT](https://opensource.org/license/MIT)

## Details

This is fictional case-linelist which contains de-indentified individual
level data on clinical outcome, demographics and region.

## References

WHO Ebola Response Team. 2014. Ebola virus disease in West Africa–the
first 9 months of the epidemic and forward projections. The New England
journal of medicine 371:1481–1495
[doi:10.1056/NEJMoa1411100](https://doi.org/10.1056/NEJMoa1411100)
