# Population data at admin level 3 for Sierra Leone (GIS basics example)

Population data at admin level 3 for Sierra Leone from Humanitarian Data
Exchange, repackaged for the GIS basics mapping example.

## Usage

``` r
gis_population_sle_admpop_adm3_2020
```

## Format

A data frame with 167 rows and 28 columns

## Source

This population data was downloaded from [Humanitarian Data
Exchange](https://data.humdata.org/dataset/cod-ps-sle). No edits were
made to this.

Within this package the original dataset is
gis_population_sle_admpop_adm3_2020.csv

This is version 1.0.

This dataset is English.
