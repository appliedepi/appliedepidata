# Raw cholera surveillance data for Viraland, 21 July 2021

Raw surveillance data for a cholera outbreak in fictional country
Viraland in 2021, with data as of 21 July.

## Usage

``` r
viraland_cholera_20210721_linelist_raw
```

## Format

A dataframe with 1476 rows and 22 columns.

## Source

Fictional data created by Applied Epi for training purposes

This is a derivate (made messy) of the clean cholera surveillance data
for Viraland, 21 July 2021.

This dataset is English.

## Details

This is a linelist with fictional case-level data from the imaginary
country Viraland, with epidemiological information only for suspected
cases of cholera. The data is intended for use alongside the
intermediate qmd course and the intro to R course.
